ROS 2 (Robot Operating System 2) is an open-source middleware framework designed for building robot applications.It is an updated version of ros1, ROS 2 is widely used in various fields including autonomous vehicles, drones, industrial automation, healthcare robotics, smart agriculture, and AI-enabled robotic vision tasks. Organizations leverage ROS 2 to accelerate research, development, and deployment of robotic solutions by utilizing its modular architecture and strong community support.


Applications:
     		* Autonomous Robots
		* Industrial Automation
		* AI and Vision Applications

how to install ros2 in ubuntu wsl, 

		1. Update Locale
			Set your locale to UTF-8:

				bash
				sudo locale-gen en_US en_US.UTF-8
				sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8
				export LANG=en_US.UTF-8
		2. Install Prerequisites
			Install essential packages for adding repositories and keys:

				bash
				sudo apt update
				sudo apt install software-properties-common curl gnupg2 lsb-release
				sudo add-apt-repository universe
		3. Add ROS 2 Repository and Keys
			Add the ROS 2 apt repository key:

				bash
				sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg
			Add the ROS 2 repository to your sources list:

				bash
				echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(lsb_release -cs) 				main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null
		4. Install ROS 2 Packages
			Update apt and install ROS 2 desktop:

			bash
			sudo apt update
			sudo apt install ros-humble-desktop
			(Replace humble with your preferred ROS 2 distribution, e.g., foxy, galactic, or jazzy.)

		5. Source ROS 2 Setup Script
			Add this to your shell startup file for convenience (~/.bashrc or ~/.zshrc):

			bash
			source /opt/ros/humble/setup.bash
			Or source it manually for the current session:

			bash
			source /opt/ros/humble/setup.bash
		6. Additional Dependencies for Building
			If you plan to build your own ROS 2 packages:

			bash
			sudo apt install python3-colcon-common-extensions python3-rosdep
			sudo rosdep init
			rosdep update
