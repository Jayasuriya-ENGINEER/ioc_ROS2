In ROS 2 (Robot Operating System 2), publishers and subscribers are fundamental parts of the topic-based communication system. Below is an explanation + a useful GitHub repository you can clone and try out.

STEPS TO BE FOLLOWED

1)Create a workspace if you haven’t already:

bash
mkdir -p ~/ros2_ws/src
cd ~/ros2_ws
colcon build
source install/setup.bash



2)Create a package
bash
cd ~/ros2_ws/src
ros2 pkg create py_pubsub --build-type ament_python --dependencies rclpy std_msgs



3)Write the Publisher node
Navigate into your package:

bash
cd ~/ros2_ws/src/py_pubsub/py_pubsub

Create a new file named publisher_member_function.py:

bash
wget wget https://raw.githubusercontent.com/ros2/examples/foxy/rclpy/topics/minimal_publisher/examples_rclpy_minimal_publisher/publisher_member_function.py




4)Write the Subscriber node
Create the subscriber file:

bash
wget wget https://raw.githubusercontent.com/ros2/examples/foxy/rclpy/topics/minimal_subscriber/examples_rclpy_minimal_subscriber/subscriber_member_function.py




5)pdate the setup.py file
Open setup.py in the py_pubsub folder and locate the entry_points section. Add both entry points:

entry_points={
    'console_scripts': [
        'talker = py_pubsub.publisher_member_function:main',
        'listener = py_pubsub.subscriber_member_function:main',
    ],
},



6)Build and run
Return to workspace root and build:

bash
cd ~/ros2_ws
colcon build
source install/setup.bash



7)Run the publisher in one terminal:

*)bash
ros2 run py_pubsub talker


*)Run the subscriber in another terminal:

bash
ros2 run py_pubsub listener
